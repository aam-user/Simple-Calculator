## Simple Arithmetic Calculator in C

A basic command-line calculator written in **C**. This project demonstrates the use of modular functions to perform fundamental arithmetic operations.

* **Modular Logic:** Each operation (Addition, Subtraction, Multiplication, Division) is handled by its own dedicated function.

* **Type Casting:** Demonstrates how to convert integers to floats for accurate division results.
* **Bilingual UI:** Prompts and outputs are in **Italian**.
